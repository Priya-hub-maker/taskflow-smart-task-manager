# TaskFlow – Smart Task Management System

## Project Description
TaskFlow is a simple Python-based task management application that helps users organize and manage their daily tasks efficiently. The system allows users to add tasks, view the task list, mark tasks as completed, and delete tasks when they are no longer needed. It also supports task priority levels such as High, Medium, and Low to help users focus on important work.

This project demonstrates how a basic software system can help improve productivity and time management by organizing tasks in a simple and structured way.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Set task priority (High / Medium / Low)

## Technologies Used
- Python
- Basic programming concepts (Lists, Loops, Functions)

## How to Run the Project
1. Install Python on your system.
2. Download the project file `taskflow.py`.
3. Open the file using a code editor like Visual Studio Code.
4. Run the program using the command:

```
python taskflow.py
```

5. Follow the menu options to manage your tasks.
